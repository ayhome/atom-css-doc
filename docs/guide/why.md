# 为什么是？

## 更小、更好、更简单
- 动态按需生成css，更小的css体积
- 大量复用css，降低项目体积
- 满足更多的样式规则，更加易用
- 可以按需自定义规则，更加个性化

## 兼容 uniapp
意味着可以编译为：h5、app、微信小程序、支付宝小程序、字节小程序... 

## ide 插件

在 hbuilderx、vscode 中提供了编辑器插件，功能：

1. 鼠标悬停查看示意
2. 代码提示
3. style 转 class
4. rpx 和 px 互换

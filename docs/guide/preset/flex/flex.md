# flex
命名规律：首字母缩写

|class		|样式						|
|--				|--							|
|flex-1		| flex: 1;			|
|f-1			| flex: 1;			|
|f-1-1		| flex: 1 1;		|
|f-1-1%		| flex: 1 1%;		|
|f-1-1-1%	| flex: 1 1 1%;	|
|f-auto		| flex: auto;		|
|f-none		| flex: none;		|

```js
class="flex-1" => "flex: 1;"
class="f-1" => "flex: 1;"
class="f-1-1" => "flex: 1 1;"
class="f-1-1%" => "flex: 1 1%;"
class="f-1-1-1%" => "flex: 1 1 1%;"
class="f-auto" => "flex: auto;"
class="f-none" => "flex: none;"
  ```
